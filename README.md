# Scaler Global — website redesign

Homepage (`index.html`) and Solutions page (`solutions.html`) built on the
Scaler marketing design system and motion system.

- `scaler-site.css` — shared chrome (nav, footer, tokens, motion) for all pages
- `colors_and_type.css` — brand design tokens (Beausite Classic + palette)
- `assets/` — logo and brand shape SVGs
- `fonts/` — Beausite Classic webfonts (licensed — confirm coverage before public deploy)
- `brandbook.html` — brand book spreads (work in progress)

All metrics shown in product illustrations are illustrative placeholders.
